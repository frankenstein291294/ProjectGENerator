## Project GENerator

Is tool to create a new proyect, focus to OOP.

## Usage 🚀
1. Clone repository
   
   ~~~
   git clone https://github.com/frankenstein291294/ProjectGENerator.git
   ~~~

2. Move the file pgen.sh to your path of projects

   ~~~
   mv pgen.sh /your-projects-path
   ~~~

3. Execute pgen.sh

   ~~~
   ./pgen.sh
   ~~~
